## Practica 04: Consumo de APIs de Redes Sociales con OAuth
---
<p align="justify">
En este proyecto se pondrán en ejemplo práctico el desarrollo una aplicación web con autenticación OAuth 2.0 para 5 redes sociales, mostrando datos del perfil del usuario después del login exitoso..</p>

---

#### Consideraciones:

<p align="justify">
El proyecto estará basado en una Arquitectura SOA (Service Oriented Architecture), el Patrón de Diseño MVC (Model, View, Controler) y servicios API REST, deberá gestionarse debidamente en el uso del contro de versiones y ramas progresivas del desarrollo del mismo.</p>
